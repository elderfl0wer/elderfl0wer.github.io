Synopsis

mathematics.h is a single-header C library covering the math I kept re-implementing across small projects — vectors, matrices, and the usual scalar helpers — without pulling in a dependency for it. Drop the header in, define the implementation macro in one translation unit, and it's available everywhere else as declarations only.

History

Started as a handful of vector/matrix helpers copy-pasted between projects until keeping them in sync got tedious. Rewritten as a proper single-header library so it could be dropped into new projects with one #include.

Usage

#define MATHEMATICS_H_IMPLEMENTATION
#include "mathematics.h"

int main(void) {
    vec3 a = { 1.0f, 2.0f, 3.0f };
    vec3 b = { 4.0f, 5.0f, 6.0f };
    vec3 c = vec3_add(a, b);
    return 0;
}
